- This is a simple test automation project covering the following scenario:
  Register user -> Login -> Logout
- This Framework was developed using Playwright, testng, selenium and java.
- This project utilizes the page object model design pattern.

Steps required to run the project:
1. clone the project
2. Inside the pom.xml run it to load all required dependencies.
3. Step into the RegisterScenario test class and click run. 
4. Observe the result.
